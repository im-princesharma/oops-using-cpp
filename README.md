## Core OOP Concepts

### Encapsulation

Combining data and methods into a single unit while restricting direct access to internal details.

### Inheritance

Acquiring properties and behaviors of an existing class into a new class.

### Polymorphism

Allowing a single interface to perform multiple tasks through overloading and overriding.

### Abstraction

Showing only essential information while hiding implementation details.
